Sobre Mí

Soy Sergio Sanchez, Técnico Universitario en Programación egresado de la UTN. Me especializo en desarrollo backend con Java y Spring Boot, construyendo APIs REST con JPA con Hibernate, Docker y MySQL.

Actualmente estoy profundizando en Spring Security y fortaleciendo mi inglés técnico. Busco oportunidades como Desarrollador Java Backend para aportar valor desde el primer día y seguir creciendo en un entorno real.
